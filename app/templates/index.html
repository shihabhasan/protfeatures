ProtFeatures uses python scripts and third-party tools to predict features from protein sequences. The ProtFeatures server can handle thousands of sequences and there is no limit for sequence number.

Input. As input, protein sequences can be uploaded or pasted in fasta format. It is recommended to provide email address for lage number of sequences. The browser window can be closed and the result will be sent by email (if provided) after finishing the job.

Output. Features for each protein are provided in a sortable table. Detail of each feature can be viewed by clicking on feature value.



Select protein type



    EukaryoticPredicts eukaryotic protein features ProkaryoticPredicts prokaryotic protein features
