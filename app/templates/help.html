SchistoProt is a supervised machine learning based tool which uses Support Vector Machines (SVMs) to discriminate between i) surface and non-surface proteins and ii) secretory and non-secretory peptides. Generated predictions are stored in a database, which facilitates the rapid reuse of predictions without rerunning the time-consuming SVM-based classification. This saves considerabe runtime if the same sequences are uploaded multiple times, e.g. by different users. SchistoProt predictions are highly valuable for identifying genes important for host-parasite interaction and the discovery of novel drug and vaccine targets.



Data & Source Code

The data set can be downloded from  here and the source codes of SchistoProt can be found on  GitHub.



Input options

SchistoProt takes the fasta formatted text or file. File extension can be any type (eg. .txt, .fas, .fasta etc.). File extension does not have any effect on SchistoProt.





If a valid email address is provided, an email containing the output result text file will be sent automatically. So for large dataset, it is better to provide the email address.



Output

After completing the task, a result page with a prediction table is appeared. The result is also sent by email if a valid email address is provided during data submission.

The prediction table is a list of Sequence ID of test protein sequence, category prediction, decision score and probability.





The feature table is a list of Sequence ID of test protein sequences and the values for 76 features.





Plots and Charts can be drawn from 'Draw Charts' section.
